search ohmibus

See [`src/ritsbus.coffee`](src/ritsbus.coffee) for full documentation.

## Installation

In hubot project repo, run:

`npm install hubot-ritsbus --save`

Then add **hubot-ritsbus** to your `external-scripts.json`:

```json
[
  "hubot-ritsbus"
]
```

## Sample Interaction

```
user1>> hubot hello
hubot>> hello!
```
